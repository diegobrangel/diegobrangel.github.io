# Política de Privacidade - Controle de Gastos

**Última atualização:** Janeiro de 2025

## Introdução

O aplicativo Controle de Gastos respeita sua privacidade e está comprometido em proteger seus dados pessoais. Esta política de privacidade explica como coletamos, usamos e protegemos suas informações.

## Dados que Coletamos

### Dados de Conta
- **Email:** Usado para autenticação e identificação da conta
- **Nome:** Exibido no perfil do usuário (opcional)
- **Foto de perfil:** Armazenada para personalização (opcional)

### Dados Financeiros
- **Gastos:** Descrição, valor, categoria, data e status de pagamento
- Todos os dados financeiros são armazenados de forma segura e associados apenas à sua conta

### Dados Técnicos
- Informações do dispositivo para melhorar a experiência do usuário
- Dados de uso anônimos para análise de desempenho

## Como Usamos seus Dados

- **Fornecer o serviço:** Permitir que você registre e gerencie seus gastos
- **Autenticação:** Garantir que apenas você tenha acesso à sua conta
- **Sincronização:** Manter seus dados sincronizados entre dispositivos
- **Melhorias:** Aprimorar a experiência do usuário

## Armazenamento e Segurança

- Seus dados são armazenados no Firebase (Google Cloud Platform)
- Utilizamos criptografia em trânsito (HTTPS/TLS)
- Autenticação segura via Firebase Authentication
- Regras de segurança que garantem que apenas você acesse seus dados

## Compartilhamento de Dados

**Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros**, exceto:
- Quando exigido por lei
- Para proteger nossos direitos legais
- Com provedores de serviço essenciais (Firebase/Google) sob contratos de proteção de dados

## Seus Direitos

Você tem o direito de:
- **Acessar** seus dados pessoais
- **Corrigir** informações imprecisas
- **Excluir** sua conta e todos os dados associados
- **Exportar** seus dados

Para exercer esses direitos, entre em contato: **diegobrangel8@gmail.com**

## Retenção de Dados

- Seus dados são mantidos enquanto sua conta estiver ativa
- Ao excluir sua conta, todos os dados são removidos permanentemente

## Menores de Idade

Este aplicativo não é destinado a menores de 13 anos. Não coletamos intencionalmente dados de crianças.

## Alterações nesta Política

Podemos atualizar esta política periodicamente. Notificaremos sobre mudanças significativas através do aplicativo.
